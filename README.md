# Auto Sales Exploratory Data Analysis (EDA)

This repository contains a comprehensive Exploratory Data Analysis (EDA) on an automotive sales dataset. The project focuses on uncovering sales trends, seasonal patterns, and geographic performance using Python.

## Project Overview
The analysis identifies key drivers of revenue by examining market activity from 2018 to 2020. It provides insights into high-performing regions, peak sales periods, and product line popularity.

## Repository Structure
* `AutoSalesEda.ipynb`: The main Jupyter Notebook containing data cleaning, statistical analysis, and visualizations.
* `Auto Sales data.csv`: The dataset containing 2,747 records of sales transactions.

## Key Insights

### 1. Revenue Trends
* **Total Sales:** The dataset accounts for a total revenue of **₹9,760,221.71**.
* **Yearly Performance:**
    * **2019:** Peak year with **₹4.67M** in sales.
    * **2018:** Strong performance with **₹3.35M**.
    * **2020:** Noticed a decline to **₹1.73M**.

### 2. Seasonality
* **Peak Month:** **November** recorded the highest sales volume, suggesting strong year-end demand.
* **Low Point:** **June** recorded the lowest sales performance.

### 3. Geographic Performance
* **Top Market:** The **USA** is the highest-grossing country.
* **Lowest Market:** The **UK** showed the lowest revenue among analyzed regions.
